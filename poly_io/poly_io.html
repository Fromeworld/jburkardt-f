POLY_IO
Read and Write POLY Files.

POLY_IO is a FORTRAN90 library which contains routines to read or write a POLY file.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Related Data and Programs:

POLY, a data directory which contains a description and examples of the POLY file format.

SHOWME, a C program which can read a POLY file and display an X Window image or create a PostScript image.

TEST_MESH, a FORTRAN90 library which defines a few test meshes.

TEST_TRIANGULATION, a FORTRAN90 library which defines a number of test regions.

TRIANGLE, a C program which computes Voronoi diagrams and Delaunay triangulations; it uses POLY files for some of its input.

Reference:

  1. http://www-2.cs.cmu.edu/~quake/triangle.html, the TRIANGLE web site;

Examples and Tests:

  • poly_io_prb.f90, a sample calling program.

  • poly_io_prb_output.txt, the output file.
  • shape.poly, a POLY file created by one of the tests.

List of Routines:

  • GET_UNIT returns a free FORTRAN unit number.
  • POLY_HEADER_READ reads header data from a POLY file.
  • POLY_WRITE writes data to a POLY file.
  • S_TO_I4 reads an I4 from a string.
  • S_WORD_EXTRACT extracts the next word from a string.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
