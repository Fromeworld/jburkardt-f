TABLE_HISTOGRAM
Histogram Table Data

TABLE_HISTOGRAM is a FORTRAN90 program which reads a table of data from a TABLE file, and computes a histogram for each column of data.

The user specifies the number of bins, and the minimum and maximum values of data to bin. This interval is divided up into equal intervals or "bins", and the number of data items in each bin is counted.

The program writes the histogram data to an output TABLE file, consisting of one column of bin center values, and a column of population counts for each column of input data.

Usage:

table_histogram table_file bin_num bin_min bin_max
where
  • table_file, the file containing the data.
  • bin_num, the number of bins.
  • bin_min, bin_max, the range of values to bin.
