CRYSTAL_PLOT
Plot CRYSTAL Output

CRYSTAL_PLOT is a FORTRAN90 program which reads in the data from a run of CRYSTAL_QED, and allows the user to select the physical quantities to plot. Options include the ability to zoom in, to set the colors of various objects, and to make a reflected copy of the data when symmetry means that only half the data was computed.
