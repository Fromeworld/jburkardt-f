G95_INTRINSICS
Examples of G95 Intrinsic Functions

G95_INTRINSICS is a FORTRAN90 program which demonstrates the use and features of some of the intrinsic routines peculiar to the G95 FORTRAN compiler.

These functions are not listed in the FORTRAN standard; they are in some cases commonly available in C or C++ compilers.

Examples and Tests:

  • g95_intrinsics.f90, a single program that calls many tests; each test is devoted to a particular G95 intrinsic function.
  • g95_intrinsics.sh, commands that compile and run the test;
  • g95_intrinsics_output.txt, the output file;

  • TEST_DERFC checks DERFC against ERFC_VALUES.

  • TEST_ERF checks ERF against ERF_VALUES.
  • TEST_ERFC checks ERFC against ERFC_VALUES.

  • TEST_SIZEOF tests SIZEOF.

  • BESSEL_J1_VALUES returns some values of the J1 Bessel function.
