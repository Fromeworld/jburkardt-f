DSP_TO_ST
Convert Sparse Matrix File from DSP to ST Format

DSP_TO_ST is a FORTRAN90 program which converts a sparse matrix file from the DSP (one-based-index) format to Sparse Triplet (ST) (zero-based index) format.

Usage:

dsp_to_st file.dsp
reads the user's data file file.dsp, reads the data, and writes the converted data to file.st.

  • kershaw.st, the Kershaw matrix, stored in an ST format file created by the command
