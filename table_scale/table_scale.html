TABLE_SCALE
Scale a Table Dataset

TABLE_SCALE is a FORTRAN90 program which adjusts an M dimensional dataset of N points by scaling each entry.

TABLE_SCALE expects the input data file to be in the simple TABLE format.

Usage:

table_scale file.txt
where
  • file.txt is the user's data file;
prompts for a scale vector from the user, and then writes the scaled data to a file which will be called file.scale.txt.

  • grid121.scale14.txt, the points, scaled by S=(1,4).
