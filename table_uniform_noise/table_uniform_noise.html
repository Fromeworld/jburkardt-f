TABLE_UNIFORM_NOISE
Add Uniform Random Noise to a Table Dataset

TABLE_UNIFORM_NOISE is a FORTRAN90 program which adjusts an M dimensional dataset of N points by adding scaled uniform "noise" to each entry.

TABLE_UNIFORM_NOISE expects the input data file to be in the simple TABLE format.

Usage:

table_uniform_noise file.txt
reads the user's data file file.txt, prompts for a scale vector and a random number seed from the user, and then writes the noisy data to a file which will be called file.uniform_noise.txt.
