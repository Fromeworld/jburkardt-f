HERMITE_POLYNOMIAL
Hermite Polynomials

HERMITE_POLYNOMIAL is a FORTRAN90 library which evaluates the physicist's Hermite polynomial, the probabilist's Hermite polynomial, the Hermite function, and related functions.

The physicist's Hermite polynomial H(n,x) can be defined by: 

        H(n,x) = (-1)^n exp(x^2/2) * d^n/dx^n ( exp(-x^2/2) )

The normalized physicist's Hermite polynomial Hn(n,x) is scaled so that 

        Integral ( -oo < X < +oo ) exp ( - X^2 ) * Hn(M,X) Hn(N,X) dX = delta ( N, M )

The probabilist's Hermite polynomial He(n,x) is related to H(n,x) by: 

        He(n,x) = H(n,x/sqrt(2)) / sqrt ( 2^n )

The normalized probabilist's Hermite polynomial Hen(n,x) is scaled so that 

        Integral ( -oo < X < +oo ) exp ( - 0.5*X^2 ) * Hen(M,X) Hen(N,X) dX = delta ( N, M )

The Hermite function Hf(n,x) is related to H(n,x) by: 

        Hf(n,x) = H(n,x) * exp(-x^2/2) / sqrt ( 2^n * n! * sqrt ( pi ) )

The Hermite function Hf(n,x) is scaled so that: 

        Integral ( -oo < X < +oo ) Hf(M,X) Hf(N,X) dX = delta ( N, M )

Related Data and Programs:

CHEBYSHEV_POLYNOMIAL, a FORTRAN90 library which evaluates the Chebyshev polynomial and associated functions.

GEN_HERMITE_RULE, a FORTRAN90 program which can compute and print a generalized Gauss-Hermite quadrature rule.

HERMITE_RULE, a FORTRAN90 program which can compute and print a Gauss-Hermite quadrature rule.

JACOBI_POLYNOMIAL, a FORTRAN90 library which evaluates the Jacobi polynomial and associated functions.

LAGUERRE_POLYNOMIAL, a FORTRAN90 library which evaluates the Laguerre polynomial, the generalized Laguerre polynomials, and the Laguerre function.

LEGENDRE_POLYNOMIAL, a FORTRAN90 library which evaluates the Legendre polynomial and associated functions.

PCE_BURGERS, a FORTRAN90 program which defines and solves a version of the time-dependent viscous Burgers equation, with uncertain viscosity, using a polynomial chaos expansion in terms of Hermite polynomials, by Gianluca Iaccarino.

PCE_ODE_HERMITE, a FORTRAN90 program which sets up a simple scalar ODE for exponential decay with an uncertain decay rate, using a polynomial chaos expansion in terms of Hermite polynomials.

POLPAK, a FORTRAN90 library which evaluates a variety of mathematical functions.

TEST_VALUES, a FORTRAN90 library which supplies test values of various mathematical functions.

List of Routines:

  • H_INTEGRAL evaluates a monomial physicist's Hermite integral for H(n,x).
  • H_POLYNOMIAL evaluates the physicist's Hermite polynomial H(n,x).
  • H_POLYNOMIAL_COEFFICIENTS: coeffs of physicist's Hermite polynomial H(n,x).
  • H_POLYNOMIAL_VALUES: values of the physicist's Hermite polynomial H(n,x).
  • H_POLYNOMIAL_ZEROS: zeros of the physicist's Hermite polynomial H(n,x).
  • H_QUADRATURE_RULE: quadrature for physicist's Hermite polynomial H(n,x).
  • H_TRIPLE_PRODUCT_INTEGRAL: integral of H(i,x)*H(j,x)*H(k,x)*e^(-x^2/2).
  • HE_INTEGRAL evaluates a monomial probabilist's Hermite integral for He(n,x).
  • HE_POLYNOMIAL evaluates the probabilist's Hermite polynomial He(n,x).
  • HE_POLYNOMIAL_COEFFICIENTS: coeffs probabilist's Hermite polynomial He(n,x).
  • HE_POLYNOMIAL_VALUES: values of the probabilist's Hermite polynomial He(n,x).
  • HE_POLYNOMIAL_ZEROS: zeros of the probabilist's Hermite polynomial He(n,x).
  • HE_QUADRATURE_RULE: quadrature for probabilist's Hermite polynomial He(n,x).
  • HEN_EXPONENTIAL_PRODUCT: probabilist Hermite exponential product, Hen(n,x).
  • HEN_POLYNOMIAL: evaluate normalized probabilist's Hermite poly Hen(n,x).
  • HEN_POWER_PRODUCT: power products, normalized probabilist's Hermite Hen(n,x).
  • HF_EXPONENTIAL_PRODUCT: exponential products, Hermite function Hf(n,x).
  • HF_FUNCTION evaluates the Hermite function Hf(n,x).
  • HF_FUNCTION_VALUES: values of the Hermite function Hf(n,x).
  • HF_POWER_PRODUCT: power products for Hermite function Hf(n,x).
  • HF_QUADRATURE_RULE: quadrature for Hermite function Hf(n,x).
  • HN_EXPONENTIAL_PRODUCT: exponential products for Hn(n,x).
  • HN_POLYNOMIAL evaluates normalized physicist's Hermite polynomials Hn(n,x).
  • HN_POWER_PRODUCT: power products for normalized physicist's Hermite Hn(n,x).
  • IMTQLX diagonalizes a symmetric tridiagonal matrix.
  • R8_FACTORIAL computes the factorial of N.
  • R8_FACTORIAL2 computes the double factorial function.
  • R8_SIGN returns the sign of an R8.
  • R8MAT_PRINT prints an R8MAT.
  • R8MAT_PRINT_SOME prints some of an R8MAT.
  • R8VEC_PRINT prints an R8VEC.
  • R8VEC2_PRINT prints an R8VEC2.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
