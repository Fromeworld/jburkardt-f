MODULE_MARK
Mark FORTRAN90 Modules

MODULE_MARK is a FORTRAN90 program which can replace a bare END statement by a more elaborate terminator, such as

END SUBROUTINE ALPHA

Usage:

module_mark input_file output_file
where
  • input_file is the name of the input file;
  • output_file is the name of the output file, to be created by the program.
