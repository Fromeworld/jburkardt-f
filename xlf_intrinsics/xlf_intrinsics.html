XLF_INTRINSICS
Examples of IBM XLF Intrinsic Functions

XLF_INTRINSICS is a FORTRAN90 program which demonstrates the use and features of some of the intrinsic routines peculiar to the IBM XLF FORTRAN compiler.

These functions are not listed in the FORTRAN standard; they are in some cases commonly available in C or C++ compilers.

Examples and Tests:

  • xlf_intrinsics_prb.f90, a single program that calls many tests; eact test is devoted to a particular XLF intrinsic function.
  • xlf_intrinsics_prb.sh, commands that compile and run the test;
  • xlf_intrinsics_prb_output.txt, the output file;

List of Routines:

  • TEST_ERF checks ERF against ERF_VALUES.
  • TEST_ERFC checks ERFC against ERFC_VALUES.
  • TEST_ETIME_ tests ETIME_.
  • TEST_FDATE_ tests FDATE_.
  • TEST_GAMMA checks GAMMA against GAMMA_VALUES.
  • TEST_HOSTNM_ tests HOSTNM_.
  • TEST_LGAMMA tests LGAMMA..
  • TEST_RAND tests RAND.
  • TEST_SIZEOF tests SIZEOF.
  • TEST_SLEEP_ tests SLEEP_.
  • TEST_SYSTEM tests SYSTEM.
  • TEST_TIME_ tests TIME_.
  • TEST_TIMEF tests TIMEF.
  • ERF_VALUES returns some values of the ERF or "error" function.
  • ERFC_VALUES returns some values of the ERFC or "complementary error" function.
  • GAMMA_VALUES returns some values of the GAMMA function.
  • GAMMA_LOG_VALUES returns some values of the log(GAMMA) function.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
